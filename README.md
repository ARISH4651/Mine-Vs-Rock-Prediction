# Rock vs Mine Prediction using Logistic Regression

This project predicts whether a sonar signal corresponds to a **Rock (R)** or a **Mine (M)** using Logistic Regression.

## Dataset
- **Name:** Sonar Mines vs Rocks Dataset
- **Source:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/datasets/connectionist+bench+sonar+mines+vs.+rocks)
- **Features:** 60 numeric attributes representing sonar signal energy at different frequencies.
- **Target:** R (Rock) or M (Mine)

## Steps in the Notebook
1. Data loading and exploration
2. Train-test split with stratification
3. Logistic Regression model training
4. Model evaluation (accuracy score, confusion matrix, classification report)
5. Predictions on new data
